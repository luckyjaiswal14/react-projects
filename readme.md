# React Project Collection

A curated collection of React learning projects built with Vite, React 18, Tailwind CSS, and modern React patterns.

## Included Projects

- `01basicreact` - A simple React starter app using Create React App conventions.
- `01basicvite` - A basic Vite + React setup with JSX, assets, and simple components.
- `02counter` - A counter app demonstrating state updates and event handling.
- `03tailwindprops` - Tailwind CSS styling with reusable components and props.
- `04bgChanger` - Dynamic background changer project using Tailwind and React state.
- `05passwordGenerator` - Password generator with length, numbers, and special character options.
- `06currencyConvertor` - Currency converter using a custom hook and remote rate data.
- `07reactRouter` - React Router demonstration for navigation and multi-page routing.
- `08miniContext` - Mini example of React Context for shared state.
- `09themeswitcher` - Theme switcher sample for light/dark mode toggling.
- `10todocontextLocal` - Todo app with context and local storage persistence.
- `customReact` - A custom React-like implementation exploration.

## How to Use

1. Open the project folder for the app you want to run.
2. Install dependencies with `npm install`.
3. Start the development server with `npm run dev`.
4. Build production assets with `npm run build`.

## Notes

- Each project is self-contained with its own `package.json`.
- Tailwind styles are configured in projects that use Tailwind CSS.
- `06currencyConvertor` fetches currency rates from a CDN API.
